# Typography

We are using InterVariable font with custom font features and numeric variants for better readability and consistency.

### Font Family

```css
font-sans:
  'InterVariable',
  system-ui,
  -apple-system,
  BlinkMacSystemFont,
  'Segoe UI',
  Roboto;
font-mono: ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, 'Liberation Mono', 'Courier New', monospace;
```

### Font Weights

- Regular: `font-regular` (440)
- Medium: `font-medium` (530)
- Semibold: `font-semibold` (660)

### Line Heights

- Tight: `leading-tight` (16px)
- Base: `leading-base` (18px)
- Loose: `leading-loose` (24px)

#### Available Text Classes

```jsx
// Extra Small - 11px
text-xs

// Small - 12px
text-sm
text-sm-bold (with medium weight)

// Base - 14px
text-base
text-base-bold (with semibold weight)
text-action (with medium weight)

// Large - 16px
text-lg (with medium weight)

// Extra Large - 18px
text-xl (with medium weight)

// 2XL - 24px
text-2xl (with medium weight)
```

### Text Decorations

We provide two utility classes for text decorations:

```jsx
// Regular underline
text-base-underline;

// Dashed underline
text-base-underline--dash;
```

### Font Features

The InterVariable font is configured with specific OpenType features:

- Font features: `'cv08', 'cv10', 'cv02'`
- Numeric variants: `lining-nums tabular-nums`
